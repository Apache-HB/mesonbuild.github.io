Downloading Meson

Meson releases can be downloaded from the Github release page. The newest development code can be obtained directly from Git

Dependencies

Meson is implemented in Python 3. If you operating system provides a package manager, use should should install it with that. For platforms that don't have a package manager, you need do download it from Python's home page.

Depending on your platform and backend you wish to use, you might need the Ninja executable. Again, use your distro-provided version if possible. Otherwise download it from here.
